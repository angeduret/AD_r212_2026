# Rapport — TD1

## Points clés

### 1. Les différentes parties d'un site web
En haut on retrouve le header, ensuite la navigation du suite, ensuite une section ou article qui créer des blocs structurer
sur la page du site internet, et tout en bas le footer

### 2. Différenciation entre HTML, CSS et Javascript
- Le html sert a la base du site : les textes, les titres, les différentes parties du site
- Le css est là pour créer un visuel au site et a créer des fletbox pour structuré le site
- Le Javascript est reconnaissable par les actions que fait le site quand l'utilisateur clique,
commme par exemple un menu qui se déroule quand on survole ou un éléments graphique qui se déplace sur la page quand on scroll
