# Rapport — TP2 Javascript

## Points clés

### 1. Aria expanded

Sert à dire l'état de l'élément : true (dans le cas de l'exercice menu ouvert) ou false (iic menu fermer)

### 2. Menu toggle

le ! du "estDejaOuverte" sert à inverser la valeur : si c'est true il se met en false, et si c'est false il se met en true
donc si le bloc FAQ était déjà ouvert au moment de cliquer, il reste fermé, sinon il s'ouvre au clic
